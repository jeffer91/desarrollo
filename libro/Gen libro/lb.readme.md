# Gen libro

Pantalla independiente para generar el libro de asignatura.

## Reglas iniciales

- Todos los archivos de esta pantalla inician con `lb`.
- La carpeta queda paralela a `carga-materia`.
- La pantalla se comunica con Carga de la materia solo por almacenamiento/base local.
- La salida final será un archivo Word automatizado.
- El formato del documento usará Candara 14.
- La tabla de contenidos debe prepararse para Word.
