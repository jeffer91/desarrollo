Certi

Generación automática de certificados institucionales de reconocimiento, capacitación y texto editable.

Modo Excel + Texto

Configuración inicial

Seleccione el tipo de certificado, período, fecha oficial y cargue los datos correspondientes.

Para capacitación, el Excel puede tener columnas como Cargo, Nombre, capacitación, Capacitador y Calificación.

Ejemplos aceptados Carrera | Estudiante | Promedio Desarrollo de Software | Juan Pérez Zambrano | 9.85 Carrera: Administración / Carlos Ruiz Mera - 9.91

Texto libre del certificado

Pegue el contenido completo. Puede usar etiquetas o escribirlo libremente; Certi separará y ordenará cada parte.

Etiquetas opcionales: TÍTULO, BENEFICIARIOS, INTRODUCCIÓN, TEXTO, DESTACADO, DETALLE, CIERRE, CIUDAD, FECHA, FIRMANTE 1, CARGO 1. Para varios certificados completos, separe cada bloque con una línea que contenga ---.
Pegue un texto y presione “Ordenar texto automáticamente”.
