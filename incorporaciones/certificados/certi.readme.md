# Módulo Certi

## 1. Objetivo

El módulo **Certi** permite generar certificados de reconocimiento para los mejores egresados por carrera dentro del sistema de **Incorporaciones**.

La mejora actual permite trabajar con dos tipos de entrada:

- Excel de mejores egresados.
- Texto pegado de mejores egresados.

La finalidad es que el usuario no dependa únicamente de un archivo Excel. Ahora también puede pegar información desde correo, WhatsApp, Word, PDF, tabla copiada o listado manual.
