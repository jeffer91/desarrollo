# Módulo Certi

## 1. Objetivo

El módulo **Certi** permite generar certificados de reconocimiento para los mejores egresados por carrera dentro del sistema de **Incorporaciones**.

La pantalla permite:

- Seleccionar un período.
- Seleccionar una fecha oficial mediante calendario.
- Cargar un Excel de mejores egresados.
- Procesar automáticamente los mejores promedios por carrera.
- Normalizar carreras con catálogo oficial.
- Detectar empates.
- Detectar carreras no reconocidas.
- Descargar un PDF único.
- Descargar PDFs individuales.

La pantalla no muestra vista previa del certificado.
