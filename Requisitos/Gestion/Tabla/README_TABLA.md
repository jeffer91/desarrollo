# Bloque 5 - Tabla

Pantalla recuperada para consultar estudiantes guardados en BaseLocal.

Flujo recomendado:
1. Entrar a Requisito.
2. Crear o seleccionar período.
3. Cargar y analizar Excel.
4. Entrar a tabla.
5. Filtrar por período, carrera, estado o búsqueda.

La tabla usa localStorage mediante ExcelLocalRepo.
No modifica Títulos.
